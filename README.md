# 智能记账本 (Budget)

一个简洁、功能完整的 HarmonyOS 本地记账应用。

## 📱 功能特性

### 核心功能
- ✅ **交易管理**：完整的增删查改 (CRUD) 操作
  - 新增收入/支出交易
  - 编辑已有交易
  - 删除单笔或多笔交易（滑动删除）
  - 查看交易列表
  
- 🏷️ **分类管理**：灵活的分类系统
  - 预设分类（不可删除）
    - 支出：餐饮、交通、购物、娱乐、医疗、教育、住房、其他
    - 收入：工资、奖金、投资收益、其他
  - 自定义分类（可添加、可删除）
  
- 🔍 **过滤和搜索**
  - 按交易类型过滤（全部/收入/支出）
  - 按分类过滤
  - 关键词搜索（搜索描述内容）

## 🏗️ 技术架构

### 数据模型
- **Transaction（交易）**
  - id: 唯一标识符
  - type: 交易类型（income/expense）
  - amount: 金额（以分为单位存储）
  - categoryId: 关联分类ID
  - description: 交易描述
  - date: 交易日期（时间戳）

- **Category（分类）**
  - id: 唯一标识符
  - name: 分类名称
  - type: 分类类型（income/expense）
  - icon: 图标名称
  - isPreset: 是否预设分类

### 数据持久化
使用 HarmonyOS Preferences API 实现本地数据存储：
- 简单的 JSON 格式存储
- 所有数据存储在本地，无需网络连接
- DataService 单例模式管理所有数据操作

### 页面结构
1. **TransactionListPage（交易列表页）**
   - 应用主页
   - 显示所有交易记录
   - 支持过滤、搜索、滑动删除
   
2. **TransactionEditPage（交易编辑页）**
   - 新增/编辑交易
   - 表单验证
   - 日期选择器
   
3. **CategoryManagePage（分类管理页）**
   - 查看所有分类
   - 添加自定义分类
   - 删除非预设分类

## 🎨 UI 设计特点

- **简洁清新**：采用绿色主题色，界面简洁易用
- **响应式交互**：滑动删除、点击编辑等流畅交互
- **信息层次清晰**：重要信息突出显示
- **色彩语义化**：
  - 收入显示绿色 (#4CAF50)

## 🚀 如何使用

### 开发环境
- DevEco Studio 5.0+
- HarmonyOS SDK API 12+

### 运行项目
1. 用 DevEco Studio 打开项目
2. 连接 HarmonyOS 设备或启动模拟器
3. 点击运行按钮

### 基本操作
1. **添加交易**：点击底部"添加交易"按钮
2. **编辑交易**：点击列表中的交易项
3. **删除交易**：向左滑动交易项，点击删除按钮
4. **过滤交易**：使用顶部的过滤按钮
5. **搜索交易**：在搜索框输入关键词
6. **管理分类**：点击右上角"分类管理"按钮

## ⚡ 性能优化

- 使用 @Observed 和 @ObjectLink 实现高效的数据绑定
- 列表项懒加载，支持大量数据
- 本地存储采用 JSON 序列化，读写效率高

## 📝 开发规范

项目严格遵循 ArkTS 开发规范：
- 强制静态类型
- 使用 `let` 代替 `var`
- 禁止使用 `any` 类型
- 使用 UIContext 获取系统 API
- 数据模型使用 `@Observed` 装饰器
